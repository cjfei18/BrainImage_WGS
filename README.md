# Deciphering the genetic architecture and regulatory programs of human brain structure and function

## Description
This repository contains the code used for the analyses presented in the manuscript 'Deciphering the genetic architecture and regulatory programs of human brain structure and function'.

The genetic association results generated in this study can be explored through our interactive web portal at [https://brain-genome-landscape.com/](https://brain-genome-landscape.com/), where users can explore associations by variant, gene, or phenotype.
